# Spec 005: Photo Journal

## Status: COMPLETE

## Summary

Allow users to take periodic photos of their plants and view a timeline of growth.

## Motivation

Watching plants grow over time is satisfying and keeps users engaged with the app. Photo journals add emotional value and retention.

## Requirements

1. Plant detail screen has an "Add Photo" button that opens camera or gallery
2. Photos are stored locally with timestamps
3. Plant detail screen shows a chronological photo timeline/gallery
4. Each photo shows the date it was taken
5. Tapping a photo opens it fullscreen with pinch-to-zoom
6. Users can delete photos from the timeline

## Acceptance Criteria

- [x] User can add a photo to a plant from camera or gallery
- [x] Photos appear in chronological order on the plant detail screen
- [x] Each photo displays its date
- [x] Tapping a photo opens fullscreen view
- [x] Fullscreen view supports pinch-to-zoom
- [x] User can delete a photo from the timeline
- [x] Photos persist across app restarts
